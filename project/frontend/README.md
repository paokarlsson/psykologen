# Psykologen Frontend

Angular-frontend för Psykologen-applikationen.

## Förutsättningar

- Node.js
- npm
- Angular CLI

## Installera dependencies

```bash
npm install
```

## Starta development server

```bash
npm start
```

Alternativt:
```bash
ng serve
```

Applikationen är tillgänglig på `http://localhost:4200/`

## Bygga för produktion

```bash
ng build
```

Build-artefakter hamnar i `dist/` mappen.

## Testa

```bash
ng test
```

## Teknologier

- Angular 20.2.0
- TypeScript 5.9.2
- RxJS 7.8.0
- Karma & Jasmine för testning
